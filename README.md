# Poker AI Pro

This is a simple web app that calculates Texas Hold’em winning probability and suggests a move.

## Live Deployment

Deploy this project on Vercel or GitHub Pages.

### Deploy on Vercel

1. Click **Deploy to Vercel**
2. Authenticate with GitHub
3. Choose this repository
4. Deploy

The app will be live at a *.vercel.app URL.

### Install on Mobile

On iPhone:
- Open the live link in Safari
- Tap **Share → Add to Home Screen**
